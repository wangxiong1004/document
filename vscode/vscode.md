- 设置成中文
```
    ctrl + shift + p -> 输入“configure language” -> 更改: "locale": "zh-cn" -> 重启vscode

    {
        "locale": "zh-cn"
    }
```

- 开启emmet：
```
    文件 -> 首选项 -> 设置 -> emmet.triggerExpansionOnTab设置为true值!

    {
        "emmet.triggerExpansionOnTab": true
    }
```
